## 사칙연산
 사칙연산이(+,-,*,/) 포함된 식을 괄호안에 넣는다.

```java 
public class Ex2_3 {
    public static void main(String[] args) {
        System.out.println(5+3);
        System.out.println(5-3);
        System.out.println(5*3);
        System.out.println(5/3);
          }
}
```
